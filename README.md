# Vietnam National Exam — English Score & Socioeconomic Analysis

## Overview
This project explores how socioeconomic conditions affect English language 
performance across Vietnam's 63 provinces, using data from the 2024 national 
university entrance exam (THPT). We merged exam score data with provincial 
statistics on GDP per capita, household computer access rate, and poverty rate 
to uncover structural disparities in English education nationwide.

## Key Findings
- Provinces with higher GDP per capita and computer access rates tend to score 
  significantly higher in English
- Poverty rate shows a negative correlation with English exam scores
- Clear regional score gaps exist between the North, Central, and South of Vietnam

## Datasets
- 2024 THPT national exam scores (English subject, ~1M student records)
- Provincial GDP per capita (GSO Statistical Yearbook 2023)
- Household computer access rate by province (GSO 2023)
- Multidimensional poverty rate by province (GSO 2023)

## Tools & Methods
- **Python** — Pandas, NumPy, Matplotlib, Seaborn
- Exploratory Data Analysis (EDA)
- Linear regression & correlation analysis
